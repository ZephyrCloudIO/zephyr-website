## Summary

The Server Tokens settings page has been redesigned to make managing tokens clearer and easier, with a better layout for creating, editing, copying, and deleting tokens.

## Redesigned Server Tokens Settings

The Server Tokens page in your organization's settings has been fully redesigned. The new layout makes it easier to see all your tokens at a glance, and every action — generating, editing, copying, or deleting a token — is now more clearly presented.

- The empty state now gives clear guidance on how to create your first server token.
- Generating and editing tokens uses a cleaner form that's easier to fill out.
- Copying a token to your clipboard is more prominent and accessible.
- The delete confirmation dialog is more descriptive about what will be removed.

## Smaller Improvements

- Addressed several dependency upgrades to keep the platform up to date.

## What This Means for Teams

- Managing server tokens is now faster and less error-prone with a cleaner, more intuitive interface.
- Teams that rely on server tokens for CI/CD and deployment integrations will find the settings page easier to navigate.
